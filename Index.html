ArronaGrid

Libreria sass para la maquetacion de sitios web

Version 0.1.5

DocumentacionDescargarGithub

¿Que es?

ArronaGrid es una libreria sass(con version css) para la maquetacion de sitios web adaptables para dispositivos moviles

Solo Layout

No añade botones, tipografias, colores ni ningun otro estilo

Solo compilas lo que necesitas

Con su version sass tu puedes compilar solo lo que necesita tu proyecto

Usa lo ultimo del estandar css

ArronaGrid usa las dos ultimas caracteristicas mas poderosas de css(Flexbox y Grid)

Es Open Source

Cuenta con licencia GPL Version 3 asi que es totalmente gratuito usalo para lo que necesites en tu proyecto
