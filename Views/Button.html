Botones

Agrega el mixin ag-Button a tu clase css o puedes agregar la clase ag-Button-css con el mismo nobre a un elemento

Version sass

.MiBoton{
    @include ag-Button($gap);
}

Version css

<p class="ag-Button-css"></p>

Ejemplo

Soy un boton

Botones Redondeados

Agrega el mixin ag-Button-Radius a tu clase css o puedes agregar la clase ag-Button-css-Radius con el mismo nobre a un elemento

Version sass

.MiBoton{
    @include ag-Button($gap);
    @include ag-Radius;
}

Version css

<p class="ag-Button-css ag-Button-css-Radius"></p>

Ejemplo

Soy un boton
