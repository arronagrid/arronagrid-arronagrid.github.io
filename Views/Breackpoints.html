Breakpoints

Los Breackpoints son los puntos de ruptura de tu dise√±o cuando el espacio disponible del viewport es superado por tu layout Estos puntos de ruptura estan definidos en un mapa de Sass en el archivo principal de ArronaGrid (ArronaGrid.scss)
$Breackpoints : (
    xs: 240px,
    s:  320px,
    m:  480px,
    st: 600px,
    t:  768px,
    l:  1024px,
    xl: 1200px
)   !default;

Si es un usuario avanzado en Sass puedes modificar los breackpoints deacuerdo a las nesecidades de su proyecto. Solo debes de declarar todo el mapa Sass $Breackpoint antes de importar ArronaGrid.
$Breackpoints : (
    Extra_Chico : 240px,
    Chico       : 320px,
    Mediano     : 480px,
    Simi-Tablet : 600px,
    Tablet      : 768px,
    Largo       : 1024px,
    Extra_Largo : 1200px
)   !default;

// importar ArronaGrid

// Si usa Prepros
@import "node_modules/@andresarronamontoya/arronagrid/ArronaGrid.scss";

// Si usa Gulp.js
@import "~@andresarronamontoya/arronagrid/ArronaGrid.scss";

Mobile First

ArronaGrid usa la filosofia de Mobile First lo que significa que para maquetar tus sitios web primero deberas de empezar desde un celular hasta llegar a una PC Ejemplo
// Creamos un contenedor flexbox con un flex-direccion:column en dispositivos cullo viewport 240px en adelantes
<article class="ag-Layout-Flex ag-Flex-c-xs"></article>

// Con la clase ag-Flex-r-xl cambiamos el valor de flex-direccion de column a row solo en dispositivos cullo viewport sea 1200px o superior
<article class="ag-Layout-Flex ag-Flex-c-xs ag-Flex-r-xl"></article>

Media Querys

ArronaGrid incluye un mixin para la creacion de media querys unicamente tienen que pasar el valor del breackpoint Ejemplo
.hola_mundo{
    color:red;
    @include MediaQuery(s){
        color:blue;
    }
}

// Compila a esto

.hola_mundo{
    color:red;
}

@media screen and (min-width:320px){
    .hola_mundo{
        color:blue;
    }
}
