Version Sass

  • O tambien puede instalar ArronaGrid desde NPM
    npm install --save-dev @andresarronamontoya/arronagrid

Compilacion mediante Prepros

  • Importe el archivo ArronaGrid.scss al archivo sass de su proyecto
    @import "node_modules/@andresarronamontoya/arronagrid/ArronaGrid.scss";

Compilacion mediante Gulp.js

  • Importe el archivo ArronaGrid.scss al archivo sass de su proyecto
    @import "~@andresarronamontoya/arronagrid/ArronaGrid.scss";
  • Para trabajar con ArronaGrid y la version 3 de gulp.js debera de instalar ests paquetes
    • gulp
    • gulp-sass
    • gulp-postcss
    • gulp-sourcemaps
    • cssnano
    • autoprefixer
    • autoprefixer core
    • node-sass-tilde-importer
  • Una vez instalados debera de crear el archivo gulpfile dependiendo de que version de gulp.js este usando
